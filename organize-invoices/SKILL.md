# 发票整理助手 (organize-invoices)

自动整理发票文件，识别类型、提取价格、重命名文件并生成统计汇总。

## 功能特性

✅ **自动解压**：提取ZIP压缩包中的发票文件
✅ **智能识别**：识别发票类型（餐饮、打车、火车、机票、住宿）
✅ **价格提取**：从发票内容中提取金额
✅ **文件重命名**：按"类型_价格.pdf"格式重命名
✅ **去重处理**：自动跳过重复的发票
✅ **格式转换**：OFD文件转换为PDF格式
✅ **统计汇总**：生成详细的分类统计报告

## 使用方法

```bash
# 在包含发票文件的目录中运行
/organize-invoices
```

或者在Claude Code中直接说："帮我整理发票"

**输出结果**：
- `整理后的发票_YYYYMMDD/` 文件夹：包含重命名后的PDF文件（文件夹名称带日期）
- `发票统计汇总_YYYYMMDD.xlsx`：Excel格式的详细统计报告（保存在整理文件夹的上一层，文件名带日期）

**支持的文件格式**：PDF、OFD、ZIP（包含发票的压缩包）

## 实现步骤

请按以下步骤执行：

### 1. 检查环境

检查当前目录是否包含发票文件（PDF、OFD、ZIP格式）。

### 2. 解压ZIP文件

如果存在ZIP文件，创建temp_unzip目录并解压所有ZIP文件。

### 3. 创建处理脚本

创建process_invoices.py脚本，包含以下功能：

**文本提取函数**：
- `extract_text_from_pdf()`: 使用PyPDF2和pdfplumber提取PDF文本
- `extract_text_from_ofd()`: 从OFD文件的XML中提取文本

**识别函数**：
- `identify_invoice_type()`: 根据关键词识别发票类型
  - 餐饮：餐饮、饭店、餐厅、烤、食品、美食
  - 打车：滴滴、打车、出租、网约车、快车、专车
  - 火车：火车、铁路、高铁、动车、车票、电子客票
  - 机票：机票、代订机票、航空、飞机、国际旅行社、携程、航班
  - 住宿：住宿、酒店、宾馆、旅馆、客栈

**价格提取函数**：
- `extract_price()`: 从文本中提取价格
  - 优先级：价税合计 > 票价 > 金额合计 > 合计（统一使用价税合计）
  - 处理数字之间的空格（PDF提取问题）
  - 对于通用¥模式，取最大值避免提取税额

**主处理函数**：
- `process_invoices()`:
  - 按文件名分组（PDF和OFD同名文件）
  - 优先使用OFD提取文本（效果更好）
  - 保存为PDF格式
  - 去重：相同类型和价格的发票只保留一份
  - 生成统计汇总

**统计函数**：
- `generate_summary()`: 生成Excel格式的发票统计汇总
  - 按类型分组统计
  - 计算总金额和小计
  - 按价格排序显示明细
  - 使用openpyxl生成带格式的Excel表格
  - 文件名增加日期标识
  - Excel保存在整理文件夹的上一层（父目录）

### 4. 运行脚本

执行process_invoices.py脚本处理所有发票。

### 5. 显示结果

向用户展示：
- 处理的发票数量
- 各类型发票统计
- 总金额
- 输出目录位置

## 技术要点

1. **PDF文本提取**：
   - 优先使用PyPDF2
   - 如果文本太短或有乱码，使用pdfplumber
   - 处理数字之间的空格问题

2. **OFD文件处理**：
   - OFD是ZIP格式，包含XML文件
   - 从Doc_0/Pages/Page_*/Content.xml提取TextCode元素
   - 使用命名空间：http://www.ofdspec.org/2016

3. **关键词匹配**：
   - 去除文本中的空格、换行、制表符
   - 同时匹配文件名和文本内容

4. **价格提取**：
   - 循环去除数字之间的空格
   - 使用正则表达式匹配多种价格格式
   - 避免提取税额等小金额

5. **去重逻辑**：
   - 使用(类型, 价格)作为唯一标识
   - 跳过重复的发票

## 依赖库

- PyPDF2: PDF文本提取
- pdfplumber: PDF文本提取（备选）
- openpyxl: 生成Excel文件
- zipfile: 解压ZIP和OFD文件
- xml.etree.ElementTree: 解析OFD的XML

## 输出

- **整理后的发票_YYYYMMDD/** 文件夹：包含重命名后的PDF文件（文件夹名称带日期）
- **发票统计汇总_YYYYMMDD.xlsx**：Excel格式的详细统计报告（保存在整理文件夹的上一层，文件名带日期）

## 注意事项

- 确保安装了PyPDF2、pdfplumber和openpyxl库
- 原始文件不会被修改或删除
- 如果价格提取失败，文件名会标记为"未知金额"
- 所有价格统一按照"价税合计"（含税总额）统计
- Excel文件包含格式化的表格，便于查看和分析
- 文件夹和Excel文件名都包含日期（YYYYMMDD格式），便于归档管理
- Excel文件保存在整理文件夹的上一层，方便查看和管理
